# 資料集描述

競賽資料包含超過 8,000 份古亞述楔形文字文本的轉寫，以及一套全面的元資料。我們為其中的一部分提供了對齊的英語翻譯。我們還提供來自近 900 份學術出版物的未處理文本，其中包含更多翻譯，你可以嘗試從中創建額外的訓練資料。

> **另請參閱：** [資料集說明](competitions_overview_zh-TW.md#資料集說明)，以了解更多關於這些轉寫中使用的格式慣例。

> **重要：** 這是一個程式碼競賽。`test.csv` 中的資料僅為虛擬資料，用於幫助你編寫解決方案。當你的提交被評分時，此範例測試資料將被完整的測試集替換。

---

## 檔案和欄位資訊

### train.csv

約 **1,500 份轉寫**，來自原始出土泥板的古亞述文本，每份都附有英語翻譯。

| 欄位 | 描述 |
|------|------|
| `oare_id` | 古亞述研究環境（OARE）資料庫中的識別碼。唯一識別每個文本。 |
| `transliteration` | 原始泥板文本的阿卡德語轉寫。 |
| `translation` | 相應的英語翻譯。 |

### test.csv

代表測試資料的小型範例集。當你的提交被評分時，此範例測試資料將被完整的測試集替換。測試資料中約有 **4,000 個句子**，來自約 **400 個獨特的文件**。

> **注意：** 雖然訓練資料在文件層級對齊翻譯，但測試資料在句子層級對齊翻譯。

| 欄位 | 描述 |
|------|------|
| `id` | 每個句子的唯一識別碼。 |
| `text_id` | 每個文件的唯一識別碼。 |
| `line_start`, `line_end` | 表示句子在原始泥板中的邊界。對文件中的句子進行排序。請注意，此欄位為 str 類型，值如 `1`、`1'` 或 `1''`。請參閱[現代書記員註記](competitions_overview_zh-TW.md#現代書記員註記)下關於行號的說明。 |
| `transliteration` | 原始泥板文本的阿卡德語轉寫。你的目標是產生相應的翻譯。 |

### sample_submission.csv

正確格式的範例提交檔案。詳見[評估頁面](competitions_overview_zh-TW.md#評估標準)。

## 補充資料

### published_texts.csv

約 **8,000 份轉寫**的古亞述文本，以及來自資料庫和博物館記錄的元資料欄位和目錄資訊，發布於 OARE 資料庫。你可以使用這些識別碼從連結的網站檢索額外資訊。這些轉寫未提供翻譯。

| 欄位 | 描述 |
|------|------|
| `oare_id` | OARE 資料庫中的識別碼，與 `train.csv` 中相同。 |
| `online transcript` | 在 DPI 網站上託管的轉寫稿 URL。 |
| `cdli_id` | CDLI 網站中的識別碼。多個 ID 以豎線 `\|` 分隔。 |
| `aliases` | 文本的其他已發布標籤（例如出版物編號、博物館 ID 等）。多個 ID 以豎線 `\|` 分隔。 |
| `label` | 文本的主要標籤名稱。 |
| `publication_catalog` | 在出版物和博物館記錄中找到的文本標籤。多個 ID 以豎線 `\|` 分隔。 |
| `description` | 文本的基本描述。 |
| `genre_label` | 分配給文本的基本類型。並非所有文本都有此資訊。 |
| `inventory_position` | 在博物館中找到的文本標籤。多個 ID 以豎線 `\|` 分隔。 |
| `online_catalog` | 耶魯收藏的 URL，包含 CC-0 元資料和圖像。 |
| `note` | 專家為評論或翻譯所做的註釋。 |
| `interlinear_commentary` | 對特定行討論文本的出版物參考。 |
| `online_information` | 大英博物館中的文本 URL（請注意這些圖像是大英博物館的版權，不屬於 CC）。並非所有文本都有此資訊。 |
| `excavation_no` | 從發掘中分配給文本的識別碼。 |
| `oatp_key` | 古亞述文本計畫（Old Assyrian Text Project）分配的識別碼。 |
| `eBL_id` | eBL 網站中的識別碼。 |
| `AICC_translation` | 首次發布的線上機器翻譯的 URL。請注意，這些翻譯大多品質很差。 |
| `transliteration_orig` | 來自 OARE 資料庫的原始文本轉寫。 |
| `transliteration` | 基於[這些格式建議](competitions_overview_zh-TW.md#格式建議)的文本轉寫清理版本。 |

### publications.csv

包含約 **880 份學術出版物的原始文本**，其中包含從古亞述語翻譯成多種現代語言的內容。這些文本是透過 OCR 和 LLM 後處理產生的。你可以嘗試提取這些翻譯並將它們與 `published_texts.csv` 中的轉寫對齊。

> **注意：** 翻譯通常以英語以外的語言呈現。

| 欄位 | 描述 |
|------|------|
| `pdf_name` | 提取文本的 PDF 檔案名稱。 |
| `page` | 給定文本出現的頁碼。 |
| `page_text` | 文章本身的文本。 |
| `has_akkadian` | 文本是否包含阿卡德語轉寫。 |

### bibliography.csv

`publications.csv` 中文本的書目資料。

| 欄位 | 描述 |
|------|------|
| `pdf_name` | 與 `publications.csv` 中對應的 ID。 |
| `title`, `author`, `author_place`, `journal`, `volume`, `year`, `pages` | 標準書目資料。 |

### OA_Lexicon_eBL.csv

此檔案包含所有古亞述語單詞的轉寫列表及其詞彙等價物（即在詞典中的查找方式）。包含的連結指向由 LMU 託管的線上阿卡德語詞典，即電子巴比倫圖書館（eBL）。

| 欄位 | 描述 |
|------|------|
| `type` | 單詞類型（例如 word、PN = 人名、GN = 地名）。 |
| `form` | 在轉寫中找到的字串文字單詞。 |
| `norm` | 規範化形式，去除連字符並標示元音長度。 |
| `lexeme` | 詞條形式，如詞典中所示。 |
| `eBL` | 電子巴比倫圖書館（eBL）中線上詞典的 URL。 |
| `I_IV` | 同音詞條的羅馬數字名稱，對應於 eBL 中的簡明阿卡德語詞典（CDA）。 |
| `A_D` | 同音詞條的字母名稱，對應於芝加哥亞述語詞典。 |
| `Female(f)` | 女性性別的名稱。 |
| `Alt_lex` | 替代規範化。 |

### eBL_Dictionary.csv

來自 eBL 資料庫的完整阿卡德語單詞詞典。它收集了 `OA_Lexicon_eBL.csv` 檔案中 eBL 的 URL 提供的資料。

### resources.csv

可能用於額外資料的資源列表。

### Sentences_Oare_FirstWord_LinNum.csv

用於在 `train.csv` 中的資料進行句子層級翻譯對齊的輔助工具。指示每個句子的第一個單詞及其在泥板上的位置。

## 建立額外訓練資料的建議工作流程

`publications.csv` 檔案包含來自近 900 份 PDF 的 OCR 輸出，從這些文本中提取翻譯是重要的第一步。在進行任何機器學習之前，需要重建和對齊訓練資料。以下是你可以遵循的簡單路徑：

### 步驟 1：找到每個文本及其翻譯

使用文件識別碼（ID、別名或博物館編號）將轉寫與 OCR 輸出中的相應翻譯配對。

### 步驟 2：將所有翻譯轉換為英語

來源翻譯可能以多種語言呈現（例如英語、法語、德語、土耳其語）。為了一致性，將所有內容轉換為英語。

### 步驟 3：建立句子層級對齊

將阿卡德語轉寫和匹配的英語翻譯分解為句子並成對對齊。這種句子層級的映射是訓練和評估機器翻譯模型最有用的格式。

> **結果：** 完成這些步驟後，你將擁有一個準備好進行機器學習的乾淨、對齊的資料集。
